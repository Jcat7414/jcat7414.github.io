Responsive Web Design Project - Tribute Page

Created by Joy Taylor in May 2020

My first project was completed after working through the modules of the 'Responsive Web Design Certification' offered via freeCodeCamp.

The focus was on creating a page with html and css elements such as main and title, divs for the image and caption and using id and class. I also experienced using codepen.io for the first time.

I enjoyed seeing my input evolve into a page, and remembering most of the element tags without having to look them up.

The page is a tribute to Niki Lauda, three-time F1 Champion. I am an F1 motor racing fan, and love being track-side where possible.

Credits: freecodecamp.org - Build a Tribute Page
