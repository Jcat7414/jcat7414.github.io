Responsive Web Design Project - Tribute Page

Created by Joy Taylor in May 2020

My first project was completed after working through the modules of the 'Responsive Web Design Certification' offered via freeCodeCamp.

The focus was on creating a page with html and css elements such as main and title, divs for the image and caption and using id and class. I also experienced using codepen.io for the first time.

I enjoyed seeing my input evolve into a page, and remembering most of the element tags without having to look them up.

The page is a tribute to Niki Lauda, three-time F1 Champion. I am an F1 motor racing fan, and love being track-side where possible.

Credits: freecodecamp.org - Build a Tribute Page

Pixel Art Project

Created by Joy Taylor in July 2020

During CoderDojo this week, a couple of the kids found this cute project. I helped them to navigate the learning opportunity, including html, css and some javascript.

That was yesterday, so this afternoon, I decided to see how much I could remember and recreate it from scratch without referring to any of the instructions.

I did fairly well until I got to the javascript, then I got stuck on the correct layout of the function.

Therefore, even though this project has all 3 languages, I am only counting it as a html/css project for myself as that was all that I could work out on my own.

I encourage you to jump in and have a play creating your own pixel art creation.

Credits: Code Club - Pixel Art (via rapberrypi.org)
